<h2> Алгоритмы </h2>

<h2> Data-дескрипторы (дескрипторы из src/model/descriptors.py кроме IsReadyDescriptor) </h2>
    <p>Геттер проверяет, что экземпляр существует и в зависимости от существования возвращает нужное поле или сам дескриптор. Сеттер проверяет, что экземпляр существует, в нём нет приватного атрибута, подчинённого данному дескриптору, тип вставляемого значения и вставляет его при прохождении необходимых проверок. Некоторые сеттеры позволяют менять атрибут после создания.</p>

<h2> Non-data-дескрипторы (IsReadyDescriptor из src/model/descriptors.py) </h2>
    <p>Имеет только геттер, который проверяет, выполнена ли задача. Не хранит результат.</p>

<h2> Атрибут с property (create_time из src/models/task.py) </h2>
    <p>Позволяет смотреть значение атрибута, но не позволяет менять его после инициализации.</p>
